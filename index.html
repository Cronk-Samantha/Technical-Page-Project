Let's Talk CSS

CSS stands for Cascading Style Sheets and is a style sheet language we use to - you guessed it - style our webpages. We are describing to the browser which HTML elements we want styled and how we want them styled. We can also use CSS to format webpages that re-size automatically depending on the device it's being viewed on!

A fundamental concept for anyone looking to get into web development:

  1. HTML is what we put on the website (titles, paragraphs, images, videos, links, etc..)
  2. CSS is how we style the stuff on the webpage (text size, background-color, text color, spacing, borders..)

Eventually, you then add JavaScript to add animation and make the webpage more interactive but that's getting ahead of myself...

Basic Syntax

A CSS rule, in its most basic form, consists of a selector and a declaration block.

  • The selector is exactly how it sounds, it is selecting which HTML elements we want to style/format.
  • The declaration block is where we declare what style/format properties we want applied to the selected elements.

Let's take a look at this in action! Below, I have put a sample of CSS code. The text h1 is our selector. We are selecting all the h1 elements on our HTML page. The text within the opening and closing curly brackets, { }, is our declaration block.

                
                    h1 { 
                        font-weight: bold;
                        background-color: green;
                    }

So, the example above is going to apply to all h1 elements and they will appear in bold with a green background color. Not too bad, right? The good news: that's all the syntax there is to CSS! The selector, followed by curly braces { }, then the declaration block. Of course, there are a lot more things you need to know before your ready to style a whole webpage but, now that we got the syntax down lets get into types of selectors.

Combinators

When working with CSS, a combinator is what we use to describe the relationship between selectors. There are four combinators we can use between simple selectors. Read through the different types below:

  1. Descendant Combinator(adding a space between selectors): This combinator is used to select elements that are a descendant of a specified element. 
    
                        div p {
                            font-style: italic;
                        }
    In this example, we are selecting all of the p elements found only within the div elements. You can also say you are only selecting the p elements that are contained within a div tag.

  2. Child Combinator(>): This combinator is used to select elements that are a direct descendant (or child) of a specified element. 
                            
                            div > p {
                                color: red;
                            }
    In the example above we are changing the font color to red for ONLY the p elements that are direct children of a div element. Something to note - the descendant combinator will select & apply to all of the p elements within a div element, even if it is nested 5 times. The child combinator on the other hand will ONLY apply to the elements that are a direct descendant, so only the p elements indented (nested) once within the specified element.

  3. Next Sibling Combinator(+): This combinator, indiciated with the addition symbol (+), selects an element that is directly after another element. 
                            
                            div + p {
                                font-weight: bold;
                            }
    In this example we are selecting the first p element that immediately follows the div element. Only the first p element after the div is going to appear bold.

  4. Subsequent Sibling Combinator(~): This combinator selects all elements that are siblings of the specified element. 
                            
                            p ~ div {
                                color: blue;
                            }
    The above rule will apply to all the div elements that come after a p tag and are siblings. Siblings meaning they fall under the same parent element and they are both nested at the same level.

CSS Float

Float is a property in CSS we can use to position and format content. One of the most common uses of the float property is when you want to wrap text around an image on page but, you can use it to position pretty much anything! There are 4 values we can set to the float property:

  • left: setting an element with 'float: left' will position the element to the left of its container
  • right: the element will position to the right of the container
  • none: the default value of float; the element/item will appear as it's layed out in the HTML page.
  • inherit: the element will be assigned the same float value as its parent element.
